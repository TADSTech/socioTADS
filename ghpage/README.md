# GitHub Pages Setup Guide

This directory contains the GitHub Pages site for SocioTADS.

## Structure

- `index.html` - Main landing page with project information
- `styles.css` - Professional styling for the site
- `manifest.json` - Web app manifest for PWA support
- `SETUP.md` - Detailed setup instructions

## Setting up the gh-pages Branch

The gh-pages branch serves the static site. To set it up:

1. Content in this `ghpage/` directory will be published to GitHub Pages
2. GitHub will automatically deploy from the `gh-pages` branch
3. The site will be available at `https://TADSTech.github.io/socioTADS/`

## Configuration in GitHub

1. Go to Repository Settings > Pages
2. Set Source to `gh-pages` branch
3. Set publishing directory to root (/)
4. Save changes

The site should be live within a few minutes.
